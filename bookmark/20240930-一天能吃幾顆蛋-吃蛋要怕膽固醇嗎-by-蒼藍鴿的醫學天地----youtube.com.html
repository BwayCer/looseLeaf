一天一顆蛋的講法哪裡來的？

概述 :: 2015 年之前的美國飲食指南裡建議說每天的膽固醇攝取不能超過 300 毫克，而一顆蛋它的膽固醇差不多就是250毫克。 也就是若吃超過一顆蛋，一天攝取的膽固醇量就會超標。
— 0:36

吃蛋不會增加膽固醇

隨著後來研究越來越深入發現：

其實血液裡面的膽固醇只有差不多 20% 是食物來源，80% 是你人體自己調控製造的。
— 1:25

概述 :: 當你膽固醇吃少一點點時，人體有可能會製造多一點點膽固醇進到血液裡面； 反之，吃得較多，人體也會降低製造的產量。
— 1:34

之後，

美國的飲食指南在2015年之後就完全的把膽固醇的攝取上限這個拿掉了。
— 1:57

這時你可能會有疑問，雖說飲食裡面的膽固醇只佔了 20%，但是吃多了還是會攝取較高的膽固醇啊？

沒錯，但是啊，

概述 :: 蛋黃裡面含有卵磷脂，它可以減少膽固醇的吸收。
— 3:04

概述 :: 雖然說蛋裡面，特別是蛋黃有豐富的膽固醇，但蛋黃裡面同時又有豐富的卵磷脂。 當你吃比較多顆蛋，可能三顆蛋四顆蛋的狀況下，其實你血液裡面的膽固醇也不會明顯上升。 這是一個研究上非常重要的點，提出來供大家參考。
— 3:22

烹調方式才是增加膽固醇的因素

概述 :: 假設你吃三顆蛋到四顆蛋，那我一律建議你要嘛吃水煮蛋、吃水波蛋、吃茶葉蛋。 這種越簡單烹調的方式越好,不要攝取過多的油脂。
— 3:57

概述 :: 今天當你煎蛋炒蛋，你會不會加油？ 就算你加的是什麼好的油，不管是橄欖油 植物油、動物油，不管是哪一種油裡面都有一定部分的飽和脂肪酸。 當飽和脂肪酸進到你身體裡面就會變成合成低密度膽固醇 LDL 的原料。
— 4:28

如此你的血液裡面的低密度膽固醇一定會上升。

概述 :: 我們抽血檢查中若壞的膽固醇 - 低密度膽固醇 LDL 如果常常高，常常超過 130，哇！這個五年後，十年後，心臟、腦部血管常常都會堵塞，就會增加心肌梗塞或中風的風險。
— 1:06

該吃幾分熟的蛋？

基本上生蛋是在營養學上是完全不建議的。
— 6:11

有時候你吃壽喜燒，只要蘸一下生蛋，只要量不多是可以接受。
— 6:15

但是生蛋有兩個點要注意：

  1. 衛生問題

    概述 :: 蛋經過母雞體內的時候可能會沾附到沙門氏桿菌 如果吃下可能會非常嚴重的吐啊、拉啊、腸胃炎、甚至敗血症…等等！
    — 6:21

  2. 阻止維生素 B7 的吸收

    概述 :: 生蛋白裡面會有阻止生物素 (維生素 B7) 吸收的物質。 當維生素 B7 不足就會有落髮，指甲長不好，各種營養不良的狀況出現。
    — 6:41

所以生蛋基本上是不建議多吃的，真的會有問題。
— 7:10

至於半熟蛋跟全熟蛋，以衛生的觀點當然還是建議吃全熟蛋比較好啦！ 但是 … 半熟蛋就是美味嘛！

概述 :: 既然半熟蛋沒有生蛋白影響維生素 B7 吸收的問題，再來就是要挑選通過驗證過可以生食的蛋，那吃半熟蛋應該就比較沒有問題。
— 7:21
